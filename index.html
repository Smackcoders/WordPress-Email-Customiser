Email Customizer Woocommerce Pro

Email Customizer Woocommerce Pro is one of the woocommerce supported plugins.It has the pre designed email template for defined order status.You can customize the pre designed templates easier. And you can preview the email template before send it.Also This plugin adds the custom order status 'Product delivered' with customizable email template.

Feature highlights

  • Customizable email templates for defined order statuses.
  • Preview option for email templates.
  • Added custom status 'Product Delivered' with customizable email template.

Steps to follow

  • Choose the email template you want to send
  • Click view button for email template preview
  • If you want to modify anything in the template then click Edit button
  • Edit the template and Save the changes
  • Now your template is ready to send

Simple UI

Email Customizer Woocommerce Pro plugin has simple UI. UI consists of list of order statuses and their template names. Each template has two buttons.They are view and edit.Plugin UI shown in below figure.

Template Preview

Choose the particular email template you want to send. If you want preview then Click the view button of the seleted template for email template preview.Then the email template will be displayed as shown in below figure.

Template Customization

If you want to modify any template then click Edit button of that particular template.Then the customization form will open.Make the changes and Save that changes. The changes made by you will be reflected in template. You can see the modified template by clicking view button of the appropriate template. The Template Customization form shown in below figure.

Added Custom Order Status(Product Delivered)

The new custom order status named as 'Product Delivered' will be added with the order statuses. This custom order status has the pre designed email template. This template can be customized easier. The added Custom Order Status shown in below figure.
